## Prerequisites

  * [Node](https://nodejs.org/)
  * [Yarn](https://yarnpkg.com/)

## Installing

```shell
  yarn
```

## Commands

```shell
  # Build project (development)
  yarn dev

  # Build project (production)
  yarn build

  # Watch js and css files
  yarn watch

  # Start dev server
  yarn serve
```
